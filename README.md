# Shippy McShipFace

A List of starships.

This project is built with React and Redux. It makes use of the [Star Wars API](https://swapi.co/documentation)

## Features
 - From SWAPI.co, renders a list of spaceships (i.e. starships)
 - All data and state is stored in Redux
 - Renders data from a paginated API endpoint, using multiple Fetch requests
 - Utilizes the JS Fetch API
 - ES6
 - Includes a 'Loading...' indicator for API calls
