# Shippy McShipFace

A List of starships.

This project is built with React and Redux. It makes use of the Star Wars API 

## Features
 - From SWAPI.co, renders a list of spaceships (i.e. starships)
 - All data and state stored in Redux
 - Each line displays ship name, model, manufacturer
 - Renders data from a paginated API endpoint
 - Every 8th ship has a grey background
 - Utilizes the JS Fetch API
 - ES6
 - Includes a 'Loading...' indicator for API calls
